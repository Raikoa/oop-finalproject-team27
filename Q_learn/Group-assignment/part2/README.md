# OOP Final Project: Multi-Agent Warehouse RL Environment (Part 3)

## Project Overview

This repository contains Part 2 of the OOP Final Project. It uses the frozen lake environment from gym with 8x8 size and is_slippery flag = true, and Qlearning, the num_episodes and max_step_per_episode is fixed
for training.

### Goal
  Successs Rate reach 70% consistantly
---

## Dependencies

The following packages are required:

- gymnasium
- numpy
- matplotlib

Install using:

    pip install gymnasium numpy matplotlib

---

## How to Run 

    python frozen_lake.py
